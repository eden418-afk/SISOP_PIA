Instalación de software en Linux

Uso de gestores de paquetes (APT, DNF, Pacman)

Los sistemas Linux utilizan gestores de paquetes para instalar, actualizar y eliminar software de manera sencilla y eficiente. Los más populares son:

  • APT (Advanced Package Tool): Utilizado principalmente en distribuciones basadas en Debian como Ubuntu. Comandos básicos incluyen:

    • sudo apt update: Actualiza la lista de paquetes disponibles.

    • sudo apt install nombre_paquete: Instala un paquete.

    • sudo apt upgrade: Actualiza los paquetes instalados a sus últimas versiones.

  • DNF (Dandified Yum): Utilizado en distribuciones como Fedora y CentOS. Comandos básicos:

    • sudo dnf update: Actualiza el sistema.

    • sudo dnf install nombre_paquete: Instala un paquete.

    • sudo dnf remove nombre_paquete: Elimina un paquete instalado.

  • Pacman: Es el gestor de paquetes de Arch Linux y sus derivados, como Manjaro. Este gestor permite instalar, actualizar y eliminar paquetes de software desde los repositorios oficiales de Arch o repositorios adicionales. Comandos básicos:

    • sudo pacman -Syu: Actualiza el sistema.

    • sudo pacman -S nombre_paquete: Instala un paquete.

    • sudo pacman -R nombre_paquete: Elimina un paquete instalado.

Estos gestores permiten una fácil instalación de software, ya que se encargan de resolver las dependencias necesarias automáticamente.

Instalación y actualización de software desde el terminal

El proceso de instalación de software en Linux puede variar ligeramente según la distribución, pero siempre sigue un patrón similar:

  1. Actualizar el sistema: Antes de instalar cualquier software, es buena práctica actualizar la lista de repositorios para asegurarse de que el sistema tiene la información más reciente.
    • En distribuciones basadas en Debian (como Ubuntu): sudo apt update && sudo apt upgrade
    • En Fedora o CentOS: sudo dnf update
    • En Arch Linux: sudo pacman -Syu
  2. Instalar un paquete específico: Se debe conocer el nombre del paquete a instalar.
    • En Ubuntu: sudo apt install nombre_paquete
    • En Fedora: sudo dnf install nombre_paquete
    • En Arch Linux: sudo pacman -S nombre_paquete
  3. Eliminar un paquete: Si ya no se necesita una aplicación, se puede eliminar fácilmente:
    • En Ubuntu: sudo apt remove nombre_paquete
    • En Fedora: sudo dnf remove nombre_paquete
    • En Arch Linux: sudo pacman -R nombre_paquete
  4. Instalar software desde archivos binarios o fuentes: Además de los gestores de paquetes, en Linux es posible instalar software descargando archivos binarios (.deb, .rpm) o compilando desde el código fuente, lo cual ofrece gran flexibilidad.
