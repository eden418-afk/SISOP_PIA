Comandos Básicos De Navegación

Los comandos de LINUX permiten al usuario controlar el sistema desde la interfaz de linea de comandos (CLI) en lugar de utilizar una interfaz con iconos. Estos son instrucciones de texto introducidas en la terminar para indicarle al sistema operativo exactamente qu es lo que se tiene que hacer. Los comandos que se ingresan en la terminal de LINUX se distinguen entre mayúsculas y minúsculas, ademas de seguir una sintaxis.

ls

Este comando lista los archivos y directorios en el directorio actual, mostrando todos los archivos y subdirectorios contenidos en su interior

Sintaxis

ls [opciones] [ruta]

Ejemplo

ls -/home Lista los archivos en /home con detalles

cd

Este comando cambia de directorio

Sintaxis

cd [directorio]

Ejemplo

pwd

Este comando muestra la ruta del directorio actual

Sintaxis

pwd

Ejemplo

pwd Muestra la ruta actual

mkdir

Este comando crea un nuevo directorio

Sintaxis

mkdir [directorio]

Ejemplo

mkdir proyectos --Crea un directorio llamado proyectos

rmdir

Este comando elimina un directorio vacio

Sintaxis

rmdir [directorio]

Ejemplo

rmdir proyectos --Elimina el directorio proyectos (si está vacío)
