Comandos Básicos De Navegación

Los comandos de LINUX permiten al usuario controlar el sistema desde la interfaz de línea de comandos (CLI) en lugar de utilizar una interfaz con iconos. Estos son instrucciones de texto introducidas en la terminar para indicarle al sistema operativo exactamente que es lo que se tiene que hacer. Los comandos que se ingresan en la terminal de LINUX se distinguen entre mayúsculas y minúsculas, además de seguir una sintaxis.

Esta categoría incluye comandos que permiten moverse entre directorios dentro del sistema de archivos de Linux. Los usuarios pueden cambiar de ubicación, ver la ruta actual y explorar el contenido de los directorios, facilitando el acceso a archivos y carpetas en la terminal.

ls

Este comando lista los archivos y directorios en el directorio actual, mostrando todos los archivos y subdirectorios contenidos en su interior

Sintaxis

ls [opciones] [ruta]

Opciones comunes

  • -l: lista los archivos en formato largo (muestra permisos, tamaño, etc.).
  • -a: muestra todos los archivos, incluidos los ocultos (archivos que comienzan con un punto).
  • -h: muestra el tamaño de los archivos en un formato legible para humanos (por ejemplo, MB, GB).
  • -R: lista el contenido de los subdirectorios de manera recursiva.

Ejemplo

ls -/home Lista los archivos en /home con detalles

cd

Este comando cambia de directorio

Sintaxis

cd [directorio]

Ejemplo

cd /var/log Cambia al directorio a /var/log

pwd

Este comando muestra la ruta del directorio actual

Sintaxis

pwd

Ejemplo

pwd Muestra la ruta actual

mkdir

Este comando crea un nuevo directorio

Sintaxis

mkdir [directorio]

Ejemplo

mkdir proyectos --Crea un directorio llamado proyectos

rmdir

Este comando elimina un directorio vacio

Sintaxis

rmdir [directorio]

Ejemplo

rmdir proyectos --Elimina el directorio proyectos (si está vacío)
