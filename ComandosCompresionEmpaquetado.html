Comandos De Compresion y Empaquetado

Los comandos de LINUX permiten al usuario controlar el sistema desde la interfaz de línea de comandos (CLI) en lugar de utilizar una interfaz con iconos. Estos son instrucciones de texto introducidas en la terminar para indicarle al sistema operativo exactamente que es lo que se tiene que hacer. Los comandos que se ingresan en la terminal de LINUX se distinguen entre mayúsculas y minúsculas, además de seguir una sintaxis.

Esta categoría agrupa comandos que sirven para comprimir y descomprimir archivos y directorios, lo que ayuda a ahorrar espacio en disco y facilita la transferencia de datos. Los archivos comprimidos también pueden agrupar varios archivos en uno solo, simplificando el manejo de grandes cantidades de datos.

tar

Este comando empaqueta o desempaqueta archivos.

Sintaxis

tar [opciones] [archivo]

Opciones comunes:

  • -c: crea un archivo nuevo.
  • -x: extrae los archivos de un archivo tar.
  • -v: muestra los archivos mientras se empaquetan o descomprimen.
  • -f [archivo.tar]: especifica el archivo tar que se va a crear o extraer.
  • -z: comprime o descomprime usando gzip.
  • -j: comprime o descomprime usando bzip2.
  • -r: añade archivos a un archivo tar existente.

Ejemplo

tar -cvf archivo.tar carpeta/ — Crea un archivo tar de la carpeta carpeta.

gzip

Este comando comprime archivos.

Sintaxis

gzip [archivo]

Opciones comunes:

  • -d: descomprime el archivo.
  • -r: comprime archivos de manera recursiva dentro de directorios.
  • -k: conserva el archivo original después de la compresión.
  • -v: muestra los detalles de la compresión.

Ejemplo

gzip archivo.txt — Comprime archivo.txt.

gunzip

Este comando descomprime los archivos.

Sintaxis

gunzip [archivo.gz]

Ejemplo

gunzip archivo.txt.gz — Descomprime archivo.txt.gz.

zip

Este comando comprime archivos en formato ZIP

Sintaxis

zip [opciones] [archivo.zip] [archivos...]

Ejemplo

zip comprimido.zip archivo1.txt archivo2.txt — Comprime los archivos archivo1.txt y 
archivo2.txt en comprimido.zip.

unzip

Este comando descomprime archivos ZIP.

Sintaxis

unzip [archivo.zip]

Opciones comunes:

  • -l: lista el contenido del archivo ZIP sin extraerlo.
  • -d [directorio]: especifica el directorio donde se extraerán los archivos.
  • -v: muestra el progreso durante la extracción.

Ejemplo

unzip comprimido.zip — Descomprime comprimido.zip.
