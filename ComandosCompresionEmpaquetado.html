Comandos De Compresion y Empaquetado

Los comandos de LINUX permiten al usuario controlar el sistema desde la interfaz de linea de comandos (CLI) en lugar de utilizar una interfaz con iconos. Estos son instrucciones de texto introducidas en la terminar para indicarle al sistema operativo exactamente qu es lo que se tiene que hacer. Los comandos que se ingresan en la terminal de LINUX se distinguen entre mayúsculas y minúsculas, ademas de seguir una sintaxis.

tar

Este comando empaqueta o desempaqueta archivos.

Sintaxis

tar [opciones] [archivo]

Ejemplo

tar -cvf archivo.tar carpeta/ — Crea un archivo tar de la carpeta carpeta.

gzip

Este comando comprime archivos.

Sintaxis

gzip [archivo]

Ejemplo

gzip archivo.txt — Comprime archivo.txt.

gunzip

Este comando descomprime los archivos.

Sintaxis

gunzip [archivo.gz]

Ejemplo

gunzip archivo.txt.gz — Descomprime archivo.txt.gz.

zip

Este comando comprime archivos en formato ZIP

Sintaxis

zip [opciones] [archivo.zip] [archivos...]

Ejemplo

zip comprimido.zip archivo1.txt archivo2.txt — Comprime los archivos archivo1.txt y archivo2.txt en comprimido.zip.

unzip

Este comando descomprime archivos ZIP.

Sintaxis

unzip [archivo.zip]

Ejemplo

unzip comprimido.zip — Descomprime comprimido.zip.
