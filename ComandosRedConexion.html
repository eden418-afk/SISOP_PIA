Comandos De Red Y Conexión

Los comandos de LINUX permiten al usuario controlar el sistema desde la interfaz de línea de comandos (CLI) en lugar de utilizar una interfaz con iconos. Estos son instrucciones de texto introducidas en la terminar para indicarle al sistema operativo exactamente que es lo que se tiene que hacer. Los comandos que se ingresan en la terminal de LINUX se distinguen entre mayúsculas y minúsculas, además de seguir una sintaxis.

Esta categoría incluye comandos que permiten configurar y analizar conexiones de red, así como acceder de manera remota a otros sistemas. Son fundamentales para la administración de redes, monitoreo de conectividad y acceso seguro a dispositivos remotos.

ping

Verifica la conectividad a una dirección IP o dominio.

Sintaxis

ping [opciones] [host]

Ejemplo

ping google.com — Envía paquetes a google.com para verificar la conectividad.

ifconfig

Muestra la configuración de red.

Sintaxis

ifconfig [interfaz]

Ejemplo

ifconfig eth0 — Muestra la configuración de la interfaz eth0.

netstat

Muestra estadísticas de red y conexiones activas.

Sintaxis

netstat [opciones]

Ejemplo

netstat -tuln — Muestra las conexiones activas y los puertos que están escuchando.

ssh

Conecta a un servidor remoto a través de SSH.

Sintaxis

ssh [usuario]@[host]

Ejemplo

ssh juan@192.168.1.10 — Conecta a un servidor con la IP 192.168.1.10 usando el usuario juan.

scp

Copia archivos entre sistemas mediante SSH.

Sintaxis

scp [archivo] [usuario]@[host]:[ruta]

Ejemplo

scp archivo.txt juan@192.168.1.10:/home/juan — Copia archivo.txt al servidor remoto.
