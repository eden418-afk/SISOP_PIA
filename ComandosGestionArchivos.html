Comandos De Gestion De Archivos

Los comandos de LINUX permiten al usuario controlar el sistema desde la interfaz de línea de comandos (CLI) en lugar de utilizar una interfaz con iconos. Estos son instrucciones de texto introducidas en la terminar para indicarle al sistema operativo exactamente que es lo que se tiene que hacer. Los comandos que se ingresan en la terminal de LINUX se distinguen entre mayúsculas y minúsculas, además de seguir una sintaxis.

Estos comandos son esenciales para manipular archivos y directorios. Permiten copiar, mover, renombrar y eliminar archivos, lo cual es fundamental para organizar y gestionar los datos en el sistema de archivos de Linux.

cp

Este comando copia archivos o directorios. Requiere pasar una ruta de origen y una de destino

Sintaxis

cp /origen/ /destino/

Opciones comunes

  • -r: copia directorios de forma recursiva.
  • -u: solo copia archivos si el destino no existe o si el archivo origen es más reciente.
  • -v: muestra los archivos a medida que se copian (modo detallado).
  • -i: pide confirmación antes de sobrescribir archivos existentes.

Ejemplo

cp /home/user/documents /backups/

mv

Este comando mueve o renombra archivos y directorios. Los archivos de la ruta origen se eliminan despues de haber sido movidos al destino

Sintaxis

mv /origen/ /destino/

Opciones comunes

  • -i: solicita confirmación antes de sobrescribir archivos.
  • -u: solo mueve el archivo si el origen es más nuevo que el destino o si el destino no existe.
  • -v: muestra los archivos a medida que se mueven.

Ejemplo

mv archivo.txt documentos/ --Mueve el archivo.txt al directorio documentos/

rm

Este comando elimina archivos o directorios. Una vez eliminados los archivos y directorios estos no se pueden recuperar.

Sintaxis

rm [opciones] [archivo]

Opciones comunes:

  • -r: elimina archivos y directorios de manera recursiva.
  • -f: fuerza la eliminación sin solicitar confirmación.
  • -i: pide confirmación antes de eliminar cada archivo.

Ejemplo

rm archivo.txt --Elimina el archivo archivo.txt

touch

Este comando crea un archivo vacio o actualiza la fecha de modificacion.

Sintaxis

touch [archivo]

Ejemplo

touch mundo.txt --Crea un archivo vacio llamado mundo.txt

chmod

Este comando cambia los permisos de archivos y directorios.

Sintaxis

chmod [permisos] [archivo]

Opciones comunes:

  • -R: cambia los permisos de archivos y subdirectorios de forma recursiva.

Ejemplo

chmod 755 script.sh --Cambia los permisos de script.sh para que el usuario tenga permisos 
de lectura, escritura y ejecución, y el grupo y otros solo lectura y ejecución.
