Comandos De Visualización De Archivos

Los comandos de LINUX permiten al usuario controlar el sistema desde la interfaz de línea de comandos (CLI) en lugar de utilizar una interfaz con iconos. Estos son instrucciones de texto introducidas en la terminar para indicarle al sistema operativo exactamente que es lo que se tiene que hacer. Los comandos que se ingresan en la terminal de LINUX se distinguen entre mayúsculas y minúsculas, además de seguir una sintaxis.

Los comandos de esta categoría permiten ver el contenido de archivos en la terminal. Son útiles para leer archivos de texto, revisar configuraciones o analizar logs, proporcionando diversas maneras de inspeccionar la información sin necesidad de abrir aplicaciones adicionales.

cat

Muestra el contenido de un archivo

Sintaxis

cat [archivo]

Ejemplo

cat archivo.txt — Muestra el contenido de archivo.txt.

more

Muestra el contenido de un archivo página por página.

Sintaxis

more [archivo]

Ejemplo

more archivo.txt — Muestra el contenido de archivo.txt una página a la vez.

less

Similar a more, pero con más opciones de navegación.

Sintaxis

less [archivo]

Ejemplo

less archivo.txt — Similar a more, pero con más opciones de navegación.

head

Muestra las primeras líneas de un archivo.

Sintaxis

head [opciones] [archivo]

Ejemplo

head -n 5 archivo.txt — Muestra las primeras 5 líneas de archivo.txt.

tail

Muestra las últimas líneas de un archivo.

Sintaxis

tail [opciones] [archivo]

Ejemplo

tail -n 5 archivo.txt — Muestra las últimas 5 líneas de archivo.txt.
