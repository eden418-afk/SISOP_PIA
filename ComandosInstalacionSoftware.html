Comandos De Instalación De Software

Los comandos de LINUX permiten al usuario controlar el sistema desde la interfaz de linea de comandos (CLI) en lugar de utilizar una interfaz con iconos. Estos son instrucciones de texto introducidas en la terminar para indicarle al sistema operativo exactamente qu es lo que se tiene que hacer. Los comandos que se ingresan en la terminal de LINUX se distinguen entre mayúsculas y minúsculas, ademas de seguir una sintaxis.

apt-get

Instala, actualiza o elimina paquetes en sistemas basados en Debian.

Sintaxis

apt-get [opciones] [paquete]

Ejemplo

sudo apt-get install vim — Instala el editor vim.

yum

Instala, actualiza o elimina paquetes en sistemas basados en Red Hat.

Sintaxis

yum [opciones] [paquete]

Ejemplo

sudo yum install httpd — Instala el servidor web Apache en sistemas basados en Red Hat.

dpkg

Gestor de paquetes para sistemas Debian, utilizado para instalar, eliminar y gestionar paquetes .deb.

Sintaxis

dpkg [opciones] [paquete]

Ejemplo

sudo dpkg -i paquete.deb — Instala el paquete .deb.

rpm

Gestiona paquetes de software en sistemas basados en Red Hat, utilizado para instalar y eliminar paquetes .rpm.

Sintaxis

rpm [opciones] [paquete]

Ejemplo

sudo rpm -ivh paquete.rpm — Instala el paquete .rpm.
